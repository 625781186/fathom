This turns a series of frozen, labeled pages into feature vectors for use with an external machine-learning system. The feature vectors land in a file in your usual downloads folder. Because web extensions aren't allowed to load file:// URLs, you might find it necessary to run a local web server like python3 -m http.server 8000 --bind 127.0.0.1 in the directory where your local samples live.
